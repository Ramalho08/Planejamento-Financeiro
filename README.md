# R💰 Ramalho Finance Enterprise V11

Projeto profissional em React + TypeScript + Vite, pronto para GitHub.

## Recursos
- Dashboard futurista
- Login local
- Firebase preparado
- Carteiras/contas
- Receitas e despesas
- Cartões e faturas
- Investimentos
- Metas financeiras
- Assistente financeiro local
- Relatórios
- Backup JSON
- Exportação CSV
- PWA preparado
- Estrutura modular

## Como rodar

```bash
npm install
npm run dev
```

## Como gerar versão final

```bash
npm run build
```

## Firebase

O arquivo `src/services/firebase.ts` já está preparado. Depois, cole suas credenciais reais do Firebase.

## GitHub

1. Crie um repositório chamado `ramalho-finance`
2. Envie todos os arquivos
3. Use GitHub Pages, Vercel ou Netlify para publicar
